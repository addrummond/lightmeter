TLV61220 (boost converter)
--------

    SW  1--- VBAT
    GND ---- VOUT
    EN  ---- FB

Pin layout with one-row DIP adapter with chip the right way up (X represents chip)::

      |
    X | - SW
    X | 
    X | - GND
    X |  
    X | - EN
    X | 
    X | - FB
    X | 
    X | - VOUT
    X | 
    X | - VBAT
      |   

Pin layout with chip upside down:

      |
    X | - FB
    X |
    X | - VOUT
    X |
    X | - VBAT
    X |
    X | - SW
    X |  
    X | - GND
    X |
    X | - EN
      |
